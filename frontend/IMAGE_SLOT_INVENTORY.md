## Notes

- All files go inside `frontend/public/` so they are served as static assets.
- If a file is missing, a soft pink placeholder with an emoji is shown — no broken image icons.
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp` (rename to `.jpg` as listed above, or ask to change the extension).
